# B-Market # 

B-Market is a Blockchain based Marketplace like eBay where a seller can add products on an open network and a buyer can purchase directly from the seller.

# Tech Stack # 
1. React JS. 
2. Truffle. 
3. Metamask. 
4. Ganache.
